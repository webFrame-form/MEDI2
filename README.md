  <h1>👨‍⚕️ 건강한 선택, MEDI:</h1>
  <h4>
    영양제의 성분을 비교하여 더 안전한 복용을 위한 의약품 정보 제공 서비스입니다.<br/><br/>
    👉🏻 주요 기능<br/><br/>
    <blockquote>
      <li>반응형 웹디자인</li>
      <li>검색/필터링</li>
      <li>모달 띄우기</li>
      <li>약 조합 계산 기능</li>
      <li>카카오 API 지도 맵 연동</li>
    </blockquote>
  </h4>
  <h4>👉🏻 실행 방법 (필독!!!!)</h4>

  1. `vscode에 medi-form 폴더 불러오기`
  2. `npm install`
  3. `npm start`
     
  <br>
